# Offline Library 1.0.0 Guide / 使用说明 / 使い方

## 中文

Offline Library 是一个 Windows 本地离线漫画和媒体资料库。它不会提供在线内容，主要用于导入你电脑里的漫画文件夹、图片、文本、音频、视频、电视剧/系列和压缩包。

### 下载和安装

1. 下载 `Offline-Library-1.0.0-x64-Setup.exe`。
2. 双击安装包。
3. 按安装向导选择安装位置和是否创建桌面快捷方式。
4. 如果电脑上已经安装过旧版，安装器会显示升级提示，并关闭正在运行的旧程序后继续升级。

### 常用功能

- 导入文件夹：导入漫画文件夹或混合资源文件夹，软件会自动分类。
- 导入 CBZ / ZIP：导入已经压缩好的漫画包。
- 搜索和筛选：可以按标题、路径、分类、收藏和 Tag 查找。
- 阅读漫画：支持方向键、空格、第一页/最后一页、缩放、适应页面、适应宽度和全屏阅读。
- 视频/音频/文本：可以使用内置查看器，也可以指定外部播放器。
- 收藏：点击爱心按钮收藏资源。
- Tag：点击标签按钮，输入逗号分隔的自定义标签。
- 高性能模式：文件很多时建议开启，并使用预加载全部内容。
- 归档和压缩：可以把已导入资源移动到指定归档目录，并把文件夹漫画压缩为 CBZ。
- 编辑者模式：可以批量选择资源，重命名外部文件/文件夹，把散图移动到新漫画文件夹，也可以删除真实磁盘文件。
- 参考图：在漫画/图片阅读器里点击“作为参考图”，会打开独立参考图窗口。即使主窗口最小化到系统托盘，参考图窗口也会保留在桌面上。
- 系统托盘：关闭主窗口会最小化到托盘；需要完全退出请点击“退出软件”或托盘菜单退出。

## English

Offline Library is a local-first Windows comic and media library. It does not provide online content. It organizes comic folders, loose images, text, audio, video, series folders, and archives already on your computer.

### Download And Install

1. Download `Offline-Library-1.0.0-x64-Setup.exe`.
2. Double-click the installer.
3. Choose the install location and shortcut options in the setup wizard.
4. If an older version is already installed, the installer shows an upgrade notice, closes the running old app, and upgrades the existing installation.

### Main Features

- Import Folder: import comic folders or mixed media folders with automatic classification.
- Import CBZ / ZIP: import compressed comic archives.
- Search and filter: find items by title, path, category, favorites, or tags.
- Comic reader: use arrow keys, Space, first/last page, zoom, fit page, fit width, and fullscreen reading.
- Video/audio/text: use the built-in viewer or configure an external player.
- Favorites: click the heart button to keep an item in Favorites.
- Tags: click the tag button and enter comma-separated custom tags.
- High performance mode: recommended for large libraries; use Preload all content for faster browsing.
- Archive and compression: move imported resources into a chosen archive folder and optionally convert folder comics to CBZ.
- Editor mode: bulk-select resources, rename external files/folders, move loose images into a new comic folder, or delete real disk files.
- Reference image: in the comic/image reader, click Use as reference. A separate reference window opens and stays visible even when the main app is minimized to the system tray.
- System tray: closing the main window minimizes it to tray. Use Quit app or the tray menu to fully exit.

## 日本語

Offline Library は Windows 向けのローカル漫画・メディアライブラリです。オンラインコンテンツは提供しません。PC 内の漫画フォルダー、画像、テキスト、音声、動画、シリーズフォルダー、圧縮ファイルを整理するためのアプリです。

### ダウンロードとインストール

1. `Offline-Library-1.0.0-x64-Setup.exe` をダウンロードします。
2. インストーラーをダブルクリックします。
3. セットアップ画面でインストール先とショートカット作成を選びます。
4. 旧バージョンがインストール済みの場合、アップグレード通知を表示し、実行中の旧アプリを閉じてから更新します。

### 主な機能

- フォルダー取り込み：漫画フォルダーや混在フォルダーを取り込み、自動分類します。
- CBZ / ZIP 取り込み：圧縮済み漫画アーカイブを取り込みます。
- 検索とフィルター：タイトル、パス、分類、お気に入り、タグで探せます。
- 漫画リーダー：方向キー、Space、最初/最後のページ、ズーム、ページ合わせ、幅合わせ、全画面読書に対応します。
- 動画/音声/テキスト：内蔵ビューアーまたは外部プレイヤーで開けます。
- お気に入り：ハートボタンでお気に入りに追加します。
- タグ：タグボタンからカンマ区切りのカスタムタグを入力します。
- 高性能モード：大量ファイルでは有効化を推奨します。すべてをプリロードすると表示が速くなります。
- 归档と圧縮：取り込み済みリソースを選択した归档フォルダーへ移動し、フォルダー漫画を CBZ に変換できます。
- 編集者モード：複数選択、外部ファイル/フォルダーの名前変更、単独画像から新しい漫画フォルダー作成、実ファイル削除ができます。
- 参考画像：漫画/画像リーダーで「参考画像にする」を押すと、独立した参考画像ウィンドウが開きます。メインアプリをシステムトレイに最小化しても残ります。
- システムトレイ：メインウィンドウを閉じるとトレイに最小化します。完全終了は終了ボタンまたはトレイメニューを使います。
